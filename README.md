# menubot
데잇걸즈 2기 선생님/운영진/교육생 모두를 위한 식사, 음주 메뉴 선택 도우미 슬랙봇


## 프로젝트 환경 세팅하기

다음 명령을 실행하여 파이썬 가상환경을 만든다.

    pipenv install

`secret.py.sample` 파일을 복사하여 `secret.py` 파일을 만들고 파이썬 토큰을 입력한다.

다음 명령으로 실행한다.

    pipenv run python main.py

프로그램을 중단하려면 `Ctrl+C` 입력.
